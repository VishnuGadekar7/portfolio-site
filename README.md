# Next.js Portfolio

This is a minimal Next.js portfolio scaffold with Tailwind CSS and Framer Motion.

Getting started:

1. Install dependencies:

   npm install

2. Start dev server:

   npm run dev

Pages:
- `/` Landing page with two pill links to `/eresume` and `/portfolio`.
- `/eresume` Minimalist eResume with embedded PDF.
- `/portfolio` Creative portfolio with project grid and animated modal.

Replace `public/pill-bg.jpg` with your background image and `public/resume.pdf` with your resume.
